Generate square wave

This module allows the generation of a square wave signal, with the possibility of modifying the parameters of amplitude, frequency, phase, active cycle and offset.

Interface

  1. Preview window: Allows to see the signal that will be generated by the module. It is updated when the value of some parameter changes.
  2. Amplitude selector: Allows to vary the amplitude of the square wave between 0 and 1. The value can be changed sliding the cursor or modifying the value in the text box.
  3. Frequency selector: Allows to vary the frequency of the square wave between 1 and 20000 Hz. The value can be changed by sliding the cursor or modifying the value in the text box.
  4. Phase selector: Allows to vary the phase of the square wave between -π and π radians. The value can be changed by sliding the cursor or modifying the value in the text box.
  5. Active cycle selector: Allows to vary the active cycle of the square wave between 10 and 90%.
  6. Duration selector: Allows to vary the duration of the signal between 1 and 30 seconds. The value can be changed by sliding the or by changing the value in the text box.
  7. Offset selector: Allows to vary the offset between -2 and 2. The value can be changed by sliding the cursor or by changing the value in the text box.
  8. "Generate" button: Opens a new display window to observe a signal with the selected parameters.
  9. Playback buttons: They allow to control the playback of the audio signal.

Square wave

A square wave is a periodic wave formed by instantaneous transitions between two levels.
The configurable parameters are as follows:

  • Amplitude: Difference between the two levels.
    In this example the amplitude is 1.
  • Frequency: Number of periods in 1 second, measured in Hz (For more information, click here).
  • Phase: Displacement of the signal with respect to its original position. (For more information, click here).
  • Active cycle: Part of the period in which the signal is at the highest level (or active level). In this case, it is measured as a percentage of the total duration of the period.

  • Offset: Constant value added to the signal. (For more information, click here).

More information

  • Frequency: Number of times an entire signal cycle is repeated along a time unit. Measured in Hertz (Hz) is the number of times a signal cycle is repeated in 1 second.
    In this example the frequency is 2 Hz
  • Phase: Displacement of the cosine. It is measured in radians, being 2π radian a complete cycle of the signal.
    In this example the signal is shifted ¼ of cycle, so the phase is π/2 rad
  • Offset: Constant value added to the signal.
    In this example the offset is 0.5
