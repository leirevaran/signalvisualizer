Control menu

This window allows to analyze the spectrum of the chosen signal by different methods: Fourier Transform, Short Time Fourier Transform, spectrogram, Short-Time-Energy, Pitch, Spectral centroid, and Filtering.

Interface

  1. Choose an option: Allows to choose the mode of visualizing the signal. The options are: FT, STFT, Spectrogram, STFT + Spectrogram, Short-Time-Energy, Pitch, Spectral centroid, and Filtering.
  2. Basic parameters: Allows to change some basic parameters: window type and length, number of points to be used in the calculation of the FFT (nfft), and overlap of consecutive windows.
  3. Spectrogram parameters: Allows to change some basic parameters of the spectrogram: its minimum and maximum frequency and its drawing style (linear/mel). It also allows to visualize the it together with the pitch (appears as a white line).
  4. Pitch parameters: Allows to change some basic parameters for the calculation of the pitch: the method for its calculation (autocorrelation/cross-correlation/subharmonics/spinet) and its minimum and maximum frequency. The "Advanced settings" button allows to change some parameters for each method.
  5. Filtering parameters: Allows to change some basic parameters for filtering the signal: the type of filter to apply (harmonic/lowpass/highpass/bandpass/bandstop), and other parameters that are only applicable depending on the chosen type of filter: the fundamental frequency multiplication, frequency of the first harmonic, percentage, and cut frequencies.
  6. Short-Time-Energy parameters: Allows to change the shape parameter for the window used in the Kaiser window type (beta).
  7. Sample frequency: Tells which is the sample frequency value of the loaded signal.
  8. "Plot" button: Allows to visualize the selected option (FT, STFT, Spectrogram...) of the signal with the values of the parameters applied.

Figure options: In every figure, there is a menu on the bottom left corner. There you can zoom and save the figure.

Fourier Transform

The Fourier Transform is a mathematical transformation used to transform signals in the time domain (in this case, the audio signal) to the frequency domain. The result shows the frequency content of the signal. In short, it is an approach to breaking down a signal into a sum of several pure tones of different frequencies.

Short Time Fourier Transform (STFT)

It consists of performing the Discrete Fourier Transform on a segment of the signal, rather than on the entire signal. This segment of the signal is obtained by multiplying the original signal by a window. In this way, only the samples of the signal inside the window will be taken into account. The values of these samples will be modified by the shape of the window (except in the case of the rectangular window).

Spectrogram

Representation that allows to see the frequency development of the audio signal over time. The horizontal axis corresponds to the time axis, the vertical axis to the frequency, and the variation of the power spectral density is represented by a colour gradient.

The spectrogram is calculated by performing several consecutive STFTs, scanning with the chosen window.
