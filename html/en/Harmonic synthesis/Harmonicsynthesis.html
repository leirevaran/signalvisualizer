Free addition of pure tones

This module allows the generation of up to six superimposed pure tones, each with a different frequency and amplitude. The keyboard at the bottom allows you to select a note to observe its first six harmonics.

Interface

  1. Frequency selector: Allows to select the frequency of each of the six components. It can be selected by typing in the text box or clicking on the arrows.
  2. Amplitude selector: Allows to select the amplitude of each of the six components. It can be selected by moving the cursor.
  3. Duration selector: Allows to vary the duration of the signal between 0.01 and 30 seconds. The value can be changed by sliding the or by changing the value in the text box.
  4. Octave: Allows to select the octave of the notes on the keyboard.
  5. "Show piano" button: Opens a piano keyboard that allows to select a note. When a note is selected, each of the six components will be assigned the frequencies of its first six harmonics, with a predetermined amplitude.
  6. "Save" button: Saves the values of the parameters. The next time openning this module, those values will be set.
  7. "Plot" button: Updates the signal (or creates one if no signal is created) with the selected parameters.
  8. Preview window: Allows to visualize the signal generated by the module.
  9. "Load" button: Opens a new control menu window to observe a signal with the selected parameters. If no fragment has been selected with the cursor, the whole signal will be loaded.

Harmonics

Harmonics are additional pure tones that arise naturally when a note is played. The frequency of each is determined by the product of the fundamental frequency (frequency of the note emitted) and a natural number (1, 2, 3, 4...).

Depending on the amplitude of each of its harmonics, the timbre of the sound changes.

Pure tone

A pure tone is a sinusoidal signal that has a single frequency. In this module it is generated using the cosine function, and it is expressed by the following formula:

A cos(2πft + 𝜙),

where A is the amplitude, f is the frequency and 𝜙 is the phase.

  • Amplitude: Maximum value of the signal.
    In this example the amplitude is 0.5
  • Frequency: Number of times an entire signal cycle is repeated along a time unit. Measured in Hertz (Hz) is the number of times a signal cycle is repeated in 1 second.
    In this example the frequency is 2 Hz
  • Phase: Displacement of the cosine. It is measured in radians, being 2π radian a complete cycle of the signal.
    In this example the signal is shifted ¼ of cycle, so the phase is π/2 rad
