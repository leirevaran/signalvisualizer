Generate sawtooth signal

This module allows the generation of a sawtooth signal, with the possibility of modifying the parameters of amplitude, frequency, phase, position of the maximum and offset.

Interface

  1. Duration selector: Allows to vary the duration of the signal between 0.01 and 30 seconds. The value can be changed by sliding the or by changing the value in the text box.
  2. Offset selector: Allows to vary the offset between -2 and 2. The value can be changed by sliding the cursor or by changing the value in the text box.
  3. Amplitude selector: Allows to vary the amplitude of the square wave between 0 and 1. The value can be changed sliding the cursor or modifying the value in the text box.
  4. Frequency selector: Allows to vary the frequency of the square wave between 1 and 20000 Hz. The value can be changed by sliding the cursor or modifying the value in the text box.
  5. Phase selector: Allows to vary the phase of the square wave between -π and π radians. The value can be changed by sliding the cursor or modifying the value in the text box.
  6. Maximum position selector: Allows to vary the position of the maximum between 0 and 1.
  7. Sample frequency: Allows to set the sample frequency of the signal that will be generated by the module. It is updated when the plot button is pressed.
  8. "Save" button: Saves the values of the parameters. The next time openning this module, those values will be set.
  9. "Plot" button: Updates the signal (or creates one if no signal is created) with the selected parameters.
  10. Scale/saturate: This option appears when the offset value is bigger than 0.5 or smaller than -0.5. The scale option converts all the bigger values than 1dB into 1, and the smaller values than -1dB into -1, whereas the saturate option divides the whole signal by the maximum absolute value of the signal if there are bigger values than 1dB, or divides it by the minimum absolute value of the signal if there are smaller values than -1dB. This will be applied to the signal if the user loads the signal.
  11. Preview window: Allows to visualize the signal generated by the module.
  12. "Load" button: Opens a new control menu window to observe a signal with the selected parameters. If no fragment has been selected with the cursor, the whole signal will be loaded.

Sawtooth signal

The sawtooth signal is a periodic signal, in which each period consists of a straight line joining an upper and a lower point. As its name suggests, it is similar to the teeth of a saw.
The configurable parameters are as follows:

  • Amplitude: Maximum signal value.
    In this example the amplitude is 1.
  • Frequency: Number of periods in 1 second, measured in Hz (For more information, click here).
  • Phase: Displacement of the signal with respect to its original position. (For more information, click here).
  • Maximum position: Maximum position within the period (as a reference, taking the minimum as the beginning of the period). 0 represents the beginning of the period, and 1 the end.
    Maximum position: 1
    Maximum position: 0.5
    Maximum position: 0
  • Offset: Constant value added to the signal. (For more information, click here).

More information

  • Frequency: Number of times an entire signal cycle is repeated along a time unit. Measured in Hertz (Hz) is the number of times a signal cycle is repeated in 1 second.
    In this example the frequency is 2 Hz.
  • Phase: Displacement of the cosine. It is measured in radians, being 2π radian a complete cycle of the signal.
    In this example the signal is shifted ¼ of cycle, so the phase is π/2 rad.
  • Offset: Constant value added to the signal.
    In this example the offset is 0.5
