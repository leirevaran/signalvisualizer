Load audio file

This module allows to load an audio file for analysis.

Interface

  1. Preview window: Allows to visualize the loaded signal.
  2. Selected fragment: Allows to select a fragment of the audio file to listen to it and to load. Appears when clicking in the waveform and dragging the mouse to the sides.
  3. “Load" button: Opens a new control menu window to observe a signal with the selected parameters. If no fragment has been selected with the cursor, the whole signal will be loaded.
